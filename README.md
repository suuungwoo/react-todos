# React Todos
Reactで作成したTo Doアプリ

## Dependency
React

## Setup

```
$ yarn
$ yarn start
```

## Usage
以下の機能があります
1. タスクが一覧できる
2. タスクが追加できる
3. タスクが完了できる
4. タスクを全て/未完了/完了でフィルタリングできる
5. タスクを削除できる
6. 完了したタスクを一括削除できる
